# KPSS Takip

KPSS Takip, Android platformu için geliştirilmiş bir mobil uygulama projesidir. Proje, Gradle (Kotlin DSL) tabanlı bir Android uygulama yapısına sahiptir ve `app` modülü üzerinden derlenmektedir. Amaç, KPSS sürecine hazırlanan kullanıcıların çalışma ve takip süreçlerini dijital ortamda yönetebileceği bir mobil altyapı sunmaktır.
